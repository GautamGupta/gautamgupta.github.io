Social It v1.5 for bbPress Released!

After a long time, I have updated the Social It plugin. It has many new features, the major one being the custom mods. Many more social bookmarking websites have also been added and some bugs have also been fixed, including the JSON compatibility bug.

Download: http://bbpress.org/plugins/topic/social-it/

Here are the changes which have been made in the 1.5 release:

  • Custom mods feature added to prevent modifications from being lost during upgrade
  • Fixed JSON compatibility issue due to multiple instances of the JSON class
  • Added Plurk, Tumblr, Strands, Stumpedia, Current, Blogger, Orkut, TheWebBlend, Wykop, BlogEngage, Hyves, Pusha, Hatena Bookmarks, MyLinkVault, SlashDot, Squidoo, Propeller, FAQpal, Evernote, Meneame, Bitacoras, JumpTags, Bebo, N4G
  • Added ability to select all, none, and popular networks
  • Removed sidebar advertisement
  • Fixed dashboard styling in Internet Explorer
  • Added a German Background Image
  • Optimized and reduced file sizes
  • Plugin now uses sprite image for (almost) all the icons in the settings page
  • Updated the screenshots

Hope you will like the 1.5 version. Please do not forget to report any bugs you face or give any feedback. Thank you! 🙂
