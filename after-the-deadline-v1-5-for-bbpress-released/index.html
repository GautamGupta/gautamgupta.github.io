After the Deadline v1.5 for bbPress Released!

After the Deadline plugin for bbPress has been updated to v1.5! There are many new things like support of multiple languages, removed the requirement of API key, and upgraded AtD/jQuery. If your BB_LANG is one of the supported languages, then it becomes the default proofreading language. It can be configured via the settings page.

Screenshots

Changelog

  • The proofreader now supports English, French, German, Portuguese and Spanish
  • Removed API key requirement
  • Updated AtD/jQuery – Changelog
  • Removed compatibility with bbPress 0.9
  • Improved coding efficiency
  • Updated translations & screenshots

Download: http://bbpress.org/plugins/topic/after-the-deadline/
