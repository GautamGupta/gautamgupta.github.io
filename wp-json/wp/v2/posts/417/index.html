Today I am releasing a new plugin for bbPress named Easy Mentions<\/a>. The plugin allows the users to link to other users’ profiles in posts by using @username<\/code>. In simple words, it is like twitter.<\/p>\n

To check it, just make a new post, write the content with a @username<\/code> (could be any username) in the text. When you submit the post, the plugin will automatically link the usernames (which exist) to their profile. Note that the plugin doesn’t link the usernames with spaces.<\/p>\n

The plugin can also add a Reply link below each post and when clicked, adds @username<\/code> with the post link<\/em> in the post text box. This could be enabled via the settings page, but before that please see #1 question in the FAQ<\/a>.<\/p>\n
