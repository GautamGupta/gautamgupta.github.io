This is a short tutorial with screenshots showing how a user can translate a WordPress or bbPress plugin (which has a POT file, most generally do) in his or her own language. Here, I have taken my bbPress plugin After the Deadline<\/a> as an example to show how to translate with a software named Poedit<\/a>. I have chosen Poedit because it has a very simple user interface, and is very easy to use.<\/p>\n

Here are the steps:<\/p>\n

    \n
  1. Download and install Poedit.<\/a><\/li>\n
  2. Open it. Go to File -> New Catalog from POT file<\/em>
    \n    <\/a><\/li>\n
  3. Select the POT file. It is generally in the root<\/em>, languages<\/em> or translations<\/em> directory of the plugin.
    \n    <\/a><\/li>\n
  4. Enter the settings in the dialog box shown.
    \n    <\/a><\/li>\n
  5. Enter the path, which is in the same dialog box. Path is the place where is the plugin stored on your local hard disk. In my case it is F:\\after-the-deadline<\/em>.<\/a><\/li>\n
  6. Write your translations. Here I have translated the plugin into the language hindi<\/em>.<\/a>Note<\/strong>: Be careful with the php values, such as %s<\/em>, %1$s<\/em> etc. Always include them in the translated string, and in the same order if it is %s<\/em> and doesn’t include a $ sign! See the screenshot above for an example. Notice that %s<\/em> is included in both strings.
    \n<\/strong><\/li>\n
  7. Save your PO file in the same directory where the POT is stored. Name it as languagecode_COUNTRYCODE<\/em>. For eg. hi_IN <\/em>(Hindi is the language and India is the country).<\/a><\/li>\n
  8. A .mo<\/em> file would also have been generated. Now notice that my POT’s name is after-the-deadline<\/em> and my PO’s name is hi_IN<\/em>. Now you have to rename the .mo<\/em> file to POT-PO.mo<\/em>. In my case, the resulted file would be after-the-deadline-hi_IN.mo<\/em>. You can do the same with the PO file too, if you wish, but it is not necessary.<\/li>\n<\/ol>\n
