Easy Twitter Links is my third plugin for bbPress. As I had told you before<\/a>, it makes your Twitter linkage easy. It automatically links @example_username to a Twitter user, and #hashtag to Twitter search link.<\/p>\n

<\/p>\n

Download: http:\/\/bbpress.org\/plugins\/topic\/easy-twitter-links\/<\/a><\/h3>\n

About<\/h2>\n

It automatically links @example_username to a Twitter user, and #hashtag to Twitter search link. For example, if you post @TheGautamGupta, it would become a link like @TheGautamGupta<\/a> and if you post #bbpress, it would become #bbpress<\/a>. The links will also have rel=”nofollow” and target=”_blank” so that your site’s link juice doesn’t get leaked into the popular social networking website Twitter . This plugin is inspired from the EZ Twitter Links<\/a> Plugin for WordPress made by Josh. Here is a screenshot below:<\/p>\n

    FAQ<\/h2>\n

    Where is the plugin options page?<\/strong><\/p>\n

    This plugin is a simple “Plug n’ Play”, and has no configurable options at the moment.<\/p>\n

    Will these links “leak” my PR?<\/strong><\/p>\n

    No. The links all use rel=”nofollow” so you won’t be leaking “link juice” to an already super popular website. (Twitter) The links also use target=”_blank”.<\/p>\n
