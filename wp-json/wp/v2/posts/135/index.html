I am glad to announce my fourth plugin for bbPress called Ajaxed Chat<\/a>. Ajaxed Chat adds a fast, customizable, multi-language Chat Room to your Forums and uses a simple file-system\/mysql storage for messages and nicknames. The plugin uses PHPFreeChat Script<\/a> as a base script to generate a chat room. It uses AJAX to smoothly refresh and display the chat zone and the nickname zone. It supports multiple rooms (`\/join`), private messages, moderation (`\/kick`, `\/ban`) and many more commands<\/a>!<\/p>\n

Download: http:\/\/bbpress.org\/plugins\/topic\/ajaxed-chat\/<\/a><\/h4>\n

Usage<\/h2>\n

You can insert the chat box anywhere in your forums by adding this code in your template files:<\/p>\n

<php if (function_exists('ajaxed_chat_load')) ajaxed_chat_load(); ?><\/pre>\n
You can also view full screen chat by going to http:\/\/example.com\/?chat where example.com is your forum link.<\/p>\n
