I am happy to announce the release of the “After the Deadline Spell Checker” plugin for bbPress. It wouldn’t have been possible if it weren’t for the excellent team behind After the Deadline<\/a>. The plugin is based on the jQuery API<\/a> of After the Deadline and can be downloaded here<\/a>.<\/p>\n

After the Deadline helps you to write better by spending less time editing. When you activate the plugin and click the link above the post textbox, the plugin will check the content for spelling, style, and grammatical errors. Please note that only English is supported for the time being. I would sincerely like to thank:<\/p>\n

Mudge, Raphael S. (2009) After the Deadline \u00e2\u20ac\u201c Language Checking Technology.<\/em> Automattic. (http:\/\/open.afterthedeadline.com<\/a>) <\/span><\/p><\/blockquote>\n

Installation<\/h2>\n

Installation is easy as 1, 2, 3, you just have to follow these steps:-<\/p>\n

1. Upload the extracted after-the-deadline<\/em> folder to the \/my-plugins\/<\/em> directory
\n2. Activate the plugin through the Plugins menu in bbPress
\n3. Enjoy and always write correct English!<\/p>\n
