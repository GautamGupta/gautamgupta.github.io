I am very happy to announce the beta release of bb Cumulus<\/a> plugin. You must have made out of the name that it has been ported from the WP Cumulus plugin by Roy Tanck and Luke Morton. bb Cumulus allows you to display your forum’s tags using a Flash movie that rotates them in a 3D sphere.<\/p>\n

bb Cumulus works just like a regular tags cloud, but is more visually exciting. Clicking the tags can be a little hard (depending on your speed setting) but does take you to the appropriate page :-).<\/p>\n

The plugin automatically replaces the generated tag heat map with the 3D cloud, but you can place it at a custom location too. Please see Other notes<\/a> for more information.<\/p>\n
