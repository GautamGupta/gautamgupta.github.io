Here is a list of the new features I am thinking to add in Social It version 1.2:<\/p>\n

    \n
  • Use your Username and API key to generate short URLs, so that you can track them<\/li>\n
  • Add bit.ly to the short URL list<\/li>\n
  • Import & Export Options<\/li>\n<\/ul>\n

    These are the features I am able to think of.<\/p>\n
