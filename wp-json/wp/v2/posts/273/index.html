Many of the users who use bbPress might have faced the problem of tags not getting deleted when they use some themes which have not been yet updated to be compatible with bbPress 1.0. In this post, we will discuss on fixing this issue.<\/p>\n

<\/a>Solution<\/h2>\n

Simply follow these 4 steps to fix the problem:<\/p>\n

    \n
  1. Open the topic-tags.php<\/em> file of your template. You may download it via a FTP program if you do not have it saved on your computer.<\/li>\n
  2. Find this code in the file:\n
    <?php if ( $public_tags ) : ?>\r\n<div id=\"othertags\">\r\n<p><?php _e('Tags:'); ?><\/p>\r\n<ul id=\"yourtaglist\">\r\n<?php foreach ( $public_tags as $tag ) : ?>\r\n<li id=\"tag-<?php echo $tag->tag_id; ?>_<?php echo $tag->user_id; ?>\"><a href=\"<?php bb_tag_link(); ?>\" rel=\"tag\"><?php bb_tag_name(); ?><\/a> <?php bb_tag_remove_link(); ?><\/li>\r\n<?php endforeach; ?>\r\n<\/ul>\r\n<\/div>\r\n<?php endif; ?><?php if ( !$tags ) : ?><\/pre>\n<\/li>\n
  3. Replace it with the following code:\n
    <p><?php _e('Tags:'); ?><\/p><?php if ( bb_get_topic_tags() ) : ?><?php bb_list_tags(); ?><?php else : ?><\/pre>\n<\/li>\n
  4. Save the file and upload it to your server. You have now successfully fixed the issue! \ud83d\ude00<\/li>\n
